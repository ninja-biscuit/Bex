07Forms

inside a div.form-group an input.form-control will display as the first example from the left in the first row. If you add the class .error to that input, will be appear like the second example (it will need by js the error message to display) in the same row. The third example in the first row is the success input, adding .success. The second row shows the same examples but with a label above.

Inside a div.form-group as a wrapper, an input.search-field will display a complete search field with a magnifier icon (on mouse over will show the pointer).
The information tooltip icon works only with bootstrap, inside a div.info-asset use a span or i element with the .icon-info class, with the attribute data-label it will display the label before the icon and the title attribute will show the tooltip content.
Checkbox and radio buttons need a wrapper as div.radio-asset and div.check-asset to generates the style change
