02Fonts

The HTML headings will take the styles automatically. You can use h1 to h6 but this style guide just set h1 to h4. Also you have preformatted text for captions and small text as declaimers. Using .number-big and .number-med classes you can highlight numbers. Each paragraph will take the style and generate a clear separation with the next object below.

03Call to action

The class .button generates the pill shape and classes like .blue-button or .gray-button gives the color variation. Use it always together. In order to add an icon, put inside any button tag, the i or span tag (empty) with the .bex-icon-pin (or whichever) class before the text.
